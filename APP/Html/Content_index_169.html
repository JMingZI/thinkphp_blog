[置顶] 由大图首页想到的

ming依然 | 2015年04月24日 17点44分 | 阅读() | 所属分类[随笔] | 网页设计 CSS3 

今天在浏览bootstrap优站时又有了新的发现，很多网站首页都习惯用一张大图，大图很清晰，但表面的文字看起来又会感觉清晰中带点模糊，使得字体变得很清晰。就像下面的效果一样，选了一张清晰的图作为背景是远远不够的：

很清晰的图


加了效果的，当然我这里加的夸张了，是为了达到这种效果：


对不对？很明显的感觉，以前自己看到这种效果的时候都是用的PS去做的，今天在看别人的源码时发现他们用的CSS3的渐变做的，为此了解了一下，把用得着的记录了一番。

渐变分为2种：线性渐变和径向渐变

常用的是线性渐变，就如效果图那样的，而径向渐变是这样的，效果图如下：


这种效果用的地方不多吧，这里我也就没看了，知道是这么回事了，用的时候再去了解吧；

下面说说线性渐变，对于IE9及以下，渐变是用的滤镜实现的，而IE10及以上便支持CSS3了，可以使用渐变了。而其它三大浏览器的渐变写法： 

/* Firefox 3.6+ */
background: -moz-linear-gradient(left, #ace, #f96);
/* Safari 5.1+, Chrome 10+ */
background: -webkit-linear-gradient(left, #ace, #f96);
/* Opera 11.10+ */
background: -o-linear-gradient(left, #ace, #f96);
从代码中可以看出，第一个是位置，这是一种简写，意思是从左到右的渐变，不简写是left right，于是就可以写很多了，top bottom等等；

另外chrome浏览器还有另外一种老式的写法 

background: -webkit-gradient(linear,top,from(#ace),to(#f96));
还可以为渐变方向加上角度，例如 

background: -moz-linear-gradient(left 30deg, #ace, #f96);

当指定的角度，请记住，它是一个由水平线与渐变线产生的的角度，逆时针方向。因此，使用0deg将产生一个左到右横向梯度，而90度将创建一个从底部到顶部的垂直渐变。






	最前面的效果图实现代码是




	

background: -webkit-linear-gradient(90deg,rgba(255,255,255,0), rgba(255,255,255,1)),url(header.jpg);

总之，我觉得这中效果肯定用得着的。
