[置顶]由新手引导特效想到的

Ming依然 | 2015年04月28日 21点22分 | 阅读() | 所属分类[随笔] | 引导特效 jscookie 

今天看了下新手引导的特效制作，没接触之前觉得很神奇，了解之后发现也不过如此。

这个引导特效的流程是这样的，当你第一次进入一个网站，它会显示这样的一个引导界面，其实每一个下一步都是一张图片，而这些图片是按照一定点击顺序显示或隐藏的，它们所处的位置，是在一个大的绝对定位的父级div层上，并且这个div层的z-index属性值高于页面中的每个元素，还有一种半透明的模糊效果，这样以来，当显示引导层的时候，用户是点击不到后面的body元素的。

还有一个知识点就是你第一次进入这个网站会写入cookie，一段时间内有效，而接下来我要说的就是js，cookie的具体内容。

tips：写入cookie是服务器行为，当你的网页没有运行在服务器环境中时，写入cookie是无效的

js对cookie的写入很简单，譬如： 

document.cookie="userId=828"; 

每个cookie都是一个名/值对，可以把下面这样一个字符串赋值给document.cookie：如果要一次存储多个名/值对，可以使用分号加空格（; ）隔开，例如：


	



document.cookie="userId=828; userName=hulk"; 



	






	另外，对于写入cookie的内容要用escape()函数进行编码，因为在cookie 的名或值中不能使用分号（;）、逗号（,）、等号（=）以及空格。





	



document.cookie="str="+escape("I love ajax"); 



	






	当然在你取值的时候，就需要解码，unescape()；




	一般我们都会设置cookie的过期时间，例如设置2天后过期：




	



//获取当前时间 
var date=new Date(); 
var expiresDays=2; 
date.setTime(date.getTime()+expiresDays*24*3600*1000); //getTime() 方法返回时间，单位为毫秒  //将userId和userName两个cookie设置为2天后过期 
document.cookie="userId=828;userName=hulk;expires="+date.toGMTString(); //GMT_String是以GMT格式表示的时间字符串 



	






	删除cookie的方法是将expireDays设置为负数，也就是一个过期了的时间。




//获取当前时间 
var date=new Date(); 
//将date设置为过去的时间 
date.setTime(date.getTime()-10000); 
//将userId这个cookie删除 
document.cookie="userId=828; expires="+date.toGMTString(); 




	






	而获取cookie，一般都是采用循环该网站下的所有cookie，遍历查找：




	



//设置两个cookie 
document.cookie="userId=828"; 
document.cookie="userName=hulk"; 
//获取cookie字符串 
var strCookie=document.cookie; 
//将多cookie切割为多个名/值对 
var arrCookie=strCookie.split("; "); 
var userId; 
//遍历cookie数组，处理每个cookie对 
for(var i=0;i<arrCookie.length;i++){ 
var arr=arrCookie[i].split("="); 
//找到名称为userId的cookie，并返回它的值 
if("userId"==arr[0]){ 
userId=arr[1]; 
break; 
} 
} 
alert(userId);
