[推荐] Thinkphp后台登录验证码不显示的问题

Ming依然 | 2015年03月25日 11点39分 | 阅读() | 所属分类[PHP] | 验证码不显示 thinkphp 

     把博客从本地上传到空间后突然发现后台登陆的验证码不显示了。

在网上找了半天解决了，也很简单。但我觉得有用就记下吧。

之前本地的代码是： 

//验证码
public function verify() {
	import('ORG.Util.Image');
	Image::buildImageVerify(4, 1, 'png', 98, 36);
}
修改为： 

//验证码
public function verify() {
	import('ORG.Util.Image');
        ob_end_clean();
	Image::buildImageVerify(4, 1, 'png', 98, 36);
}
至于为什么要加这个，应该是在你的网站中可能有很多图片要输出到缓冲区，而这个函数的作用是清除缓冲区的内容，并将缓冲区关闭，但不会输出内容。因为thinkphp自动开启了缓存。

可以了解一下ob_start(),ob_get_contents(),ob_end_clean()函数的作用。
